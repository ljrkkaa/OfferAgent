# Khoj Interview Personal KB Agent

This branch is trimmed for an initial personal knowledge base demo.

## Scope

- Obsidian sync
- Markdown, PDF, and plaintext indexing
- Chat with RAG citations
- Search, agents, tools, memory, and eval workflows

## Intentionally Removed Or Hidden

- Android, Desktop, and Emacs clients
- Documentation site, keeping only `documentation/README.md`
- Stripe billing, Twilio/phone login, S3 uploads, and external telemetry
- Speech-to-text, text-to-speech, and creative image generation entrypoints
- Notion/GitHub content source UI and API
- DOCX, image OCR, and Org-mode exposure in the first milestone

Database models and historical migrations are intentionally left in place for now.
